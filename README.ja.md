# Port Scanner by VL_PLAY Games

- [Read in English](README.md)
- [Read in Russian](README.ru.md)

## 概要

**Port Scanner** はネットワークの分析と対話のための多機能ユーティリティです。IPアドレスの確認、ポートスキャン、pingテスト、内蔵ターミナルなどを提供します。直感的なグラフィカルインターフェースで、開発者、ネットワーク管理者、パワーユーザーに最適です。

**現在のバージョン: 2.0.0**（Windows専用）。  
Linuxユーザーはバージョン **1.0.1** をご利用ください（[Releases](../../releases) から入手可能）。

## 機能

### 主な機能

- **IPの確認** – ローカルマシンのすべてのIPv4およびIPv6アドレスを表示
- **全ポートスキャン** – ローカルIP上の全ポート（1～49151）をスキャン
- **全情報** – ネットワークの包括的な概要：
  - ローカルIPアドレス
  - 各インターフェースで開いているポート
  - ネットワークアダプターの詳細設定
- **カスタムIPとポート** – 指定したIPアドレスとポート範囲をスキャン
- **Ping** – 任意のホストへの接続性と遅延を測定
- **ターミナル** – アプリ内からシステムコマンドを実行
- **アクティブデバイス** – ローカルサブネット内のデバイスを検出（IP、MACアドレス、ホスト名）

### 追加機能

- **ヘルプ** – 使用方法と機能の説明
- **ログ** – プログラムのリアルタイムログを表示
- **クリア** – 出力パネルを消去
- **設定** – ログレベルと言語（7言語対応）の変更
- **について** – バージョン情報と作者

## インターフェース

ウィンドウは2つのパネルに分割されています：

- **左パネル** – 主要機能ボタンと補助操作ボタン
- **右パネル** – 結果、ログ、プロンプトが表示される出力領域

## システム要件

- **Windows 7 / 8 / 10 / 11**
- **.NET Framework 4.7.2** 以上

## インストールと実行

### Windows（バージョン2.0.0）

1. [Releases](../../releases) ページから `port_scanner.exe` をダウンロードします。
2. `port_scanner.exe` を実行するだけです（インストール不要）。

### Linux

1. [Releases](../../releases) にある旧バージョン **1.0.1** をご利用ください。
2. `port_scanner_linux.zip`をダウンロードします
3. 内容を解凍し、ターミナルで以下を実行します:
   ```bash
   ./port_scanner
   ```
