## 概要
**Port Scanner**アプリケーションは、ネットワークの分析と対話のために設計された多目的ツールです。IPアドレスの確認、ポートのスキャン、サーバーのping、ログの表示、ターミナルコマンドへのアクセスなどの機能を提供します。直感的なグラフィカルインターフェースは、開発者、ネットワーク管理者、技術愛好家のニーズに応える多数の機能をサポートしています。

## 機能

### 🔍 主な機能
1. **IPチェック** - コンピュータのすべてのIPアドレスを表示
2. **ポートスキャン** - 開いているポートをスキャンして表示
3. **全情報** - ネットワークの完全な概要:
   - IPアドレス
   - 開いているポート
   - ネットワークドライバー
   - 追加のネットワークデータ
4. **カスタムIP** - 特定のIPアドレスのポートをスキャン
5. **Ping** - 接続性とレイテンシーの確認
6. **ターミナル** - 組み込みコマンドラインインターフェース
7. **アクティブデバイス** - ローカルネットワーク内のデバイス一覧

### ⚙️ 追加機能
- **ヘルプ** - 使用方法の説明とガイダンス
- **ログ** - プログラムの活動履歴
- **クリア** - 結果とログの消去
- **設定** - アプリケーションの設定
- **について** - バージョン情報とクレジット

## インターフェース
インターフェースは2つの主要セクションに分かれています:
- **オプション選択**: 各主要機能に対応するボタン
- **結果パネル**: 選択された操作の結果と出力を表示

## 使用方法
1. 対応するボタンをクリックして、メインメニューから機能を選択します
2. 「結果」パネルで結果やログを表示します
3. 下部の追加ボタンを使用して、ログの管理、結果の消去、ヘルプメニューへのアクセスを行います

## インストール
### Windowsの場合
1. **Releases**セクションに移動します
2. `port_scanner_windows.zip`をダウンロードします
3. 内容を解凍し、`port_scanner.exe`を実行します

### Linuxの場合
1. **Releases**セクションに移動します
2. `port_scanner_linux.zip`をダウンロードします
3. 内容を解凍し、ターミナルで以下を実行します:
   ```bash
   ./port_scanner
   ```
