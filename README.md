## Overview

**Port Scanner** is a multifunctional network utility designed for analyzing and interacting with network environments. It offers IP address discovery, port scanning, ping testing, built‑in terminal access, and more. The clean graphical interface makes it suitable for developers, system administrators, and power users.

**Current version: 2.0.0** (Windows only).  
Linux users should use version **1.0.1** (available in the [Releases](../../releases) section).

## Features

### Main Functions

- **Check your IP** – Display all IPv4 and IPv6 addresses of the local machine.
- **Check all ports** – Scan common ports (1–49151) on all local IP addresses.
- **All info** – Comprehensive network summary:
  - Local IP addresses
  - Open ports on each interface
  - Detailed network adapter configuration
- **Custom IP and ports** – Scan a user‑specified IP address and port range.
- **Ping** – Test connectivity and measure latency to any host or IP.
- **Terminal** – Execute system commands directly from the application.
- **Active devices** – Discover devices in the local subnet (IP, MAC address, hostname).

### Additional Tools

- **Help** – Usage guide and feature overview.
- **Log** – View real‑time program logs.
- **Clear** – Clear the output panel.
- **Settings** – Adjust log level and interface language (7 languages supported).
- **About** – Version and author information.

## Interface

The window is split into two panels:

- **Left panel** – Main function buttons and auxiliary controls.
- **Right panel** – Output area where results, logs, and prompts are displayed.

## System Requirements

- **Windows 7 / 8 / 10 / 11**
- **.NET Framework 4.7.2** or higher

## Installation & Usage

### Windows (version 2.0.0)

1. Download `port_scanner.exe` from the [Releases](../../releases) page.
2. Run `port_scanner.exe` – no installation required.

### Linux

1. Use the older **1.0.1** release, available on the [Releases](../../releases) page.
2. Download `port_scanner_linux.zip`.
3. Extract the contents and execute in the terminal:
   ```bash
   ./port_scanner
   ```
