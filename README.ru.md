## Обзор
Приложение **Port Scanner** - это многофункциональный инструмент для анализа и взаимодействия с сетями. Оно предоставляет функции проверки IP-адресов, сканирования портов, пинга серверов, просмотра логов и доступа к терминальным командам. Интуитивно понятный графический интерфейс поддерживает множество функций для разработчиков, сетевых администраторов и технических энтузиастов.

## Функции

### 🔍 Основные функции
1. **Проверка IP** - Отображение всех IP-адресов компьютера
2. **Сканирование портов** - Сканирование и отображение всех открытых портов
3. **Вся информация** - Полный обзор сети:
   - IP-адреса
   - Открытые порты
   - Сетевые драйверы
   - Дополнительные сетевые данные
4. **Свой IP** - Сканирование портов на определенном IP-адресе
5. **Ping** - Проверка connectivity и задержки
6. **Терминал** - Встроенный интерфейс командной строки
7. **Активные устройства** - Список устройств в локальной сети

### ⚙️ Дополнительные функции
- **Помощь** - Инструкции по использованию и руководство
- **Лог** - История активности программы
- **Очистка** - Очистка результатов и логов
- **Настройки** - Конфигурация приложения
- **О программе** - Информация о версии и разработчиках

## Интерфейс
Интерфейс разделен на две основные секции:
- **Выбор опции**: Кнопки для каждой основной функции.
- **Панель результатов**: Отображает результаты и выводы выбранной операции.

## Как использовать
1. Выберите функцию из главного меню, нажав соответствующую кнопку.
2. Просматривайте результаты или логи в панели "Результат".
3. Используйте дополнительные кнопки внизу для управления логами, очистки результатов или доступа к меню помощи.

## Установка
### Для Windows
1. Перейдите в раздел **Releases**.
2. Скачайте `port_scanner_windows.zip`.
3. Разархивируйте содержимое и запустите `port_scanner.exe`.

### Для Linux
1. Перейдите в раздел **Releases**.
2. Скачайте `port_scanner_linux.zip`.
3. Разархивируйте содержимое и выполните в терминале:
   ```bash
   ./port_scanner
    ```
