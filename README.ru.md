## Обзор

**Port Scanner** – многофункциональная сетевая утилита для анализа и взаимодействия с сетью. Позволяет просматривать IP-адреса, сканировать порты, проверять доступность узлов, использовать встроенный терминал и многое другое. Простой графический интерфейс подойдёт разработчикам, системным администраторам и опытным пользователям.

**Текущая версия: 2.0.0** (только для Windows).  
Пользователям Linux рекомендуется использовать версию **1.0.1** (см. раздел [Releases](../../releases)).

## Функции

### Основные возможности

- **Проверить IP** – отображение всех IPv4 и IPv6 адресов компьютера.
- **Проверить все порты** – сканирование портов (1–49151) на всех локальных IP.
- **Вся информация** – полная сводка по сети:
  - локальные IP-адреса
  - открытые порты для каждого интерфейса
  - детальная конфигурация сетевых адаптеров
- **Кастомный IP и порты** – сканирование заданного диапазона портов на указанном IP.
- **Ping** – проверка доступности и измерение задержки до любого хоста.
- **Терминал** – выполнение системных команд прямо из приложения.
- **Активные устройства** – поиск устройств в локальной подсети (IP, MAC, имя хоста).

### Дополнительные инструменты

- **Помощь** – руководство по использованию.
- **Лог** – просмотр журнала работы программы в реальном времени.
- **Очистить** – очистка панели вывода.
- **Настройки** – выбор уровня логирования и языка интерфейса (поддерживается 7 языков).
- **О программе** – информация о версии и авторе.

## Интерфейс

Окно разделено на две панели:

- **Левая панель** – кнопки основных функций и вспомогательных действий.
- **Правая панель** – область вывода результатов, логов и подсказок.

## Системные требования

- **Windows 7 / 8 / 10 / 11**
- **.NET Framework 4.7.2** или новее

## Установка и запуск

### Windows (версия 2.0.0)

1. Скачайте файл `port_scanner.exe` со страницы [Releases](../../releases).
2. Запустите `port_scanner.exe` – установка не требуется.

### Linux

1. Используйте предыдущую версию **1.0.1**, доступную в [Releases](../../releases).
2. Скачайте `port_scanner_linux.zip`.
3. Разархивируйте содержимое и выполните в терминале:
   ```bash
   ./port_scanner
    ```
